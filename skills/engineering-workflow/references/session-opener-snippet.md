# Session Opener Snippet

A very short operational snippet for an executing agent to use at the start of an engineering task.

## Short opener

```text
Before touching code:
1. Identify task type
2. Mark affected surfaces
3. Identify source of truth and main consumers
4. Check for public behavior impact
5. Choose Lean or Full mode
6. Decide minimum artifacts and evidence
```

## Recommended usage

- Recite once internally after loading the `engineering-workflow` skill.
- If the task is ambiguous, use this snippet for session framing first.
- If the task is small, use it to quickly decide whether Lean mode is sufficient.
