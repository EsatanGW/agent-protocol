# Phase 7: Delivery

## Goal

Produce the completion report, assemble the handoff, and list the files to commit plus any follow-ups.

## Lean / Full minimums

### Lean minimum
- Delivery summary.
- Evidence summary.
- Follow-up / risk note.

### Full minimum
- Completion report.
- Handoff narrative.
- Files to commit.
- Suggested commit message.

## Required delivery output

- Affected surfaces.
- Changed-behavior summary.
- Verification summary.
- Evidence summary.
- Known risks / follow-ups.
- Files to commit.
- Suggested commit message.
